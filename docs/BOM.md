# Bill of Materials (BOM)

| Item | Qty | Link/Notes |
|------|-----|------------|
| 120mm High Speed PWM Fan | 1 | (e.g., Delta electronics) |
| DC-DC Step Down Converter | 1 | LM2596 or similar |
| PWM Fan Speed Controller | 1 | 5V-35V 5A range |
| 18V Power Tool Battery | 1 | TPCell or DeWalt/Milwaukee |
| 18V Battery Adapter/Clip | 1 | 3D Printed or Injection Molded |
| Rocker Switch (KCD1) | 1 | SPST |
| Activated Carbon Filter | 1 | 120mm x 120mm |
| HEPA Filter Sheet | 1 | Custom Cut |
| M4 Bolts (30mm) | 4 | To secure fan |
| M3 Screws/Heat Shrink | 1 set | For internal wiring |

## Assembly Notes

1. **Wiring**: Connect the 18V battery outlet to the switch.
2. **Regulation**: Input the switched 18V into the Buck converter, set the output to 12V (or 24V depends on fan).
3. **Control**: Feed the regulated DC into the PWM controller.
4. **Mounting**: Use the 3D-printed base and grill to secure the fan and filters.
