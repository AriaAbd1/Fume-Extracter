## Assembly Notes

1. **Power**: This build is powered via a direct DC input (12V-24V) into the PWM controller.
2. **Control**: The Noctua 24V fan is controlled via the PWM dial on the speed controller.
3. **Mounting**: Use the 50mm button head screws and square nuts to secure the fan and housing.
4. **Pivot**: The steel rod serves as the main pivot for the adjustable housing.
